### About Me

I'm a software engineering student based in Tuxtla Gutiérrez, Chiapas, Mexico.  
I focus on building robust, scalable systems and understanding the **"why"** behind every line of code (First Principles), rather than just making things work.

I’m especially interested in:
- **Backend Architecture** (Hexagonal, Clean Arch)
- **Scalable Full Stack integration**
- **Code Quality & Refactoring**
- **Continuous learning** through official documentation
